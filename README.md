# 감정 기반 음악 추천 웹 어플리케이션
> Mood-based recommendation system


## 목표
표정 인식을 통한 사용자 감정 기반 음악 추천 웹 어플리케이션 개발


## 서비스 소개
- 음악추천 시스템을 활용한 감정 치유
- 다양한 감정변화에 따른 음악추천 시스템 제공
- 사용자의 음악적 취향을 고려한 맞춤형 콘텐츠 제공

## 프로젝트 FLOW
> WBS를 통한 효율적인 업무 분업화


### 음악 추천 알고리즘 API 설계 및 개발
1. luxand API 테스트
2. VA 평면 구현
3. Emotion Vector 추출
4. 감정 벡터 활용 Top 10 Music list 추천
5. 알고리즘 통합 및 테스트


### 웹 어플리케이션 개발
1. 페이지 구성 회의
2. 프론트 개발
3. 서버 개발


### 사용자 추천 알고리즘 개발
1. 추천 알고리즘 회의 및 선정
2. 서버 개발
3. 프론트 개발


### 프로젝트 마무리
1. 구현 기능 통합
2. AWS 배포
